Сервер написан на Node.js с использованием Express
Для хранения данных используется MongoDB.
Сервер задеплоен на heroku

Все файлы сервера находяться в папке api.

Папка api содержит:

1. Папка films - в ней лежат все файлы, которые управляют запросами связанными с фильмами.
2. server.js - файл, который инициализирует и настраивает сервер.

В server.js мы инициализируем сервер, мидлвары, роуты, подключаемся к базе данных MongoDB.

В папке films находиться 3 файла.

film.controller - файл, который непосредствно руководит логикой сервера. Какие данные принимаються, что с этими данными нужно сделать, какие данные отправить в ответ.

film.model создаёт модель данных.

film.router определяет, какую операцию необходимо произвести в зависимости от роута, по которому пришел запрос.
Далее будет список с методами запросов, роутом, какие данные принимает, что делает, какие данные отдаёт

1. метод: РОSТ
   роут: /films/
   принимает: объект с полями, в соответствии со схемой;
   выполняет: валидацию объекта, добавление объекта в базу данных;
   отдаёт: добавленный объект.
2. метод: GET
   роут: /films/
   принимает: -
   выполняет: достаёт из БД массив фильмов;
   отдаёт: массив с объектами фильмов.
3. метод: GET
   роут: /films/:id
   принимает: id;
   выполняет: находит в БД объект фильма, id которого совпадает с принятым id;
   отдаёт: найденный объект с фильмом.
4. метод: DELETE
   роут: /films/:id
   принимает: id;
   выполняет: находит в БД объект фильма, id которого совпадает с принятым id, и удаляет его;
   отдаёт: массив с оставшимися объектами фильмов.
